# AGENTS

This repository accepts AI-assisted contributions.

## Guardrails
- Keep changes small and reviewable.
- Prefer deterministic logic for finance-critical behavior.
- Never introduce hidden write/payment behavior.
- Do not commit secrets, tokens, or personal data.

## Required Human Checks
- Review every AI-generated change before merge.
- Validate auth, reconciliation, and payment safety paths carefully.
- Ensure docs/config changes match implementation.

## Attribution
A concise note such as "AI-assisted" in PR description is recommended for transparency.
